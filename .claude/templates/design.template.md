## 6. 에러 처리

### 6.1 예상 에러 및 처리

| 에러 | 원인 | 처리 방법 |
|-----|------|----------|
| FileNotFoundError | 파일 없음 | `st.error()` 표시 |
| APIError | API 호출 실패 | 재시도 또는 에러 메시지 |
